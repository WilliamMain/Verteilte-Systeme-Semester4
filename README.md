In semester 3 we were tasked in building a small website.

Following this, in semester 4, we were supposed to expand this website into a distributed system. 
This meant using multiple platforms/systems that can communicate with each other.
This project included using the framework xammp as a local host server, an sql database and mainly using php instead of html and js. Also, we were learning to use MVC components which is why the project is structured as it is. 
This project (which was done by me and one other student) was a challenge as we both had never used most of the new concepts we were taught.
However, we managed to work our way through it and had some fun whilst doing it. Our professor was quite laid back which led us to add some of our own flair, making some design concepts of the project a bit unrealistic, however the primary educational purpose of the project was definitely understood, because once set up correctly, the system worked quite well and was scored accordingly (in the range of 1-2 of the German educational scoring system)

All in all, it was a fun and educational project.
