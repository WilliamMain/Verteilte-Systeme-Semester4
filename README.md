In semester 3 we were tasked in biulding a small website.

Following this, in semester 4, we were supposed to expand this website into a distributed system. 
This meant using multiple platforms/systems that can communicate with each other.
This project included using the framework xammp as a local host server, an sql database and mainly using php instead of html and js. Also we were learning to use MVC conponents which is why the project is structured as it is. 
This project (which was done by me and one other student) was by all means a challenge as we both had never used most of the new concepts we were taught.
However we managed to work our way through it and had some fun whilst doing it. Our professor was and is very laid back which led us to add some of our own flair, making some design concepts of the project a bit silly, however the primary teaching of it definitley made its way through, because once set up correctly, the system worked quite well and was scored accordingly (I believe it was in the range of 1-2 of the german educational scoring system)
All in all it was a fun and educational project.

For context, WillyRoller is the semester 3 website, and markomarko was the B2B extension.
